# Release Notes

## GuapClock Desktop Beta 0.1.0

Initial free desktop beta for producer testing.

Included:

- Large 2001-style rackmount LED timer.
- Analog clock option.
- Locked artist and producer session names.
- Artist/session folder creation.
- Invoice text file creation.
- Invoice viewer panel.
- Configurable invoice destination.
- Release screenshots.
- Simple user instructions.
- Focused security review.

Beta terms:

- Free for producers to test.
- Producers who contribute useful beta feedback or improvements may receive a free license if the app launches commercially.
- This release is desktop only. No WordPress plugin is included.
