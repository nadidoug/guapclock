# GuapClock Simple Instructions

This is a free desktop beta for producers testing booth-time tracking.

## Start A Booth Session

1. Open `RUN-BOOTH-TIME-CLOCK.bat`.
2. Press `SESSION IN`.
3. Type the artist name.
4. Type the producer name.
5. Press OK.

The names are now locked for the session.

## End A Booth Session

1. Press `SESSION OUT`.
2. The app records the stop time.
3. The invoice text file is updated with total booth time.

## Find Invoices

1. Open the `Invoice` menu.
2. Choose `Show invoice panel`.
3. Select an invoice to preview it.

## Change Invoice Destination

1. Open the `Invoice` menu.
2. Choose `Invoice destination`.
3. Pick the folder where invoice text files should be saved.

## Change Music Session Folder

1. Open the `Folder` menu.
2. Choose the default music folder option.
3. Pick the folder where artist/session folders should be created.
